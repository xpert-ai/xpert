# 插件

## 开发阶段接入插件

开发阶段新增插件时，请同步更新以下位置：

- 在内置 `preBootstrapPlugins` 列表中声明插件。
- 在 `apps/api/package.json` 的 `dependencies` 中加入插件包。
- 在 `.deploy/api/package-prod.json` 的 `dependencies` 中加入插件包。

## 部署阶段接入插件

部署阶段通过 `PLUGINS` 环境变量声明插件。

- 多个插件使用英文逗号分隔。
