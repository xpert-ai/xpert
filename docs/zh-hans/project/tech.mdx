## 4. Orchestrator 设计（核心）

---

## 4.1 调度模型

```text
Swimlane = Queue
Orchestrator = Scheduler
```

---

## 4.2 调度循环

```ts
while (true) {
  lanes.sort(by priority)

  for (lane of lanes) {
    if (lane.canRun()) {
      task = lane.dequeue()
      dispatch(task)
    }
  }
}
```

---

## 4.3 执行约束

```ts
lane.canRun() {
  return (
    running < concurrencyLimit &&
    inProgress < wipLimit
  )
}
```

---

## 4.4 Task 分发

```ts
dispatch(task) {
  agent = selectAgent(task)
  env = selectEnvironment(task)

  execute(task, agent, env)
}
```

---

## 5. Agent 调度策略

```ts
selectAgent(task) {
  return agentPool
    .filter(a => a.role == task.lane.role)
    .sort(by load, successRate)
}
```

---

## 6. Environment 调度

```ts
selectEnvironment(task) {
  return environmentPool.find(e =>
    e.type == lane.environment &&
    e.available
  )
}
```
